前端开发——JS（一）

6种基本数据类型、判断语句、循环语句、iterable类型、函数的相关知识、高阶函数、闭包……

前端开发——JavaScript

一、概述

在HTML中加载JS

  1. 方式一，在ＨＴＭＬ中通过script标签，将JS代码包裹在内，推荐在外层包裹一层，确保HTML先加载完毕

  2. 方式二，通过script标签引入外部js文件

    async关键字，如果脚本无需等待页面解析，且无依赖独立运行，那么应使用 async

    defer关键字，如果脚本需要等待页面解析，且依赖于其它脚本，调用这些脚本时应使用 defer，将关联的脚本按所需顺序置于 HTML 中

基本数据类型

  • number（数字），不区分整数/浮点数，支持加减乘除求余，特殊NaN值，自己不等于自己，要通过isNaN()判断
  • string（字符串），可以用单引号、可以用双引号
  • boolean（布尔值），true和false
  • null（空值），表示非对象
  • undfined（未定义的值），表示未赋值的初始化值
  • object（对象），由键值对组成的无序集合，值可以是任何基本类型，表示复合结构的数据集

字符串

  • 不限单、双引号，如果字符串内部有引号，要加转义字符\作为前缀
  • 多行字符串，一种方式是用\n换行，另一种是ES6新加的，反引号`
  • 字符串之间可以通过+拼接，也可以通过反引号 hello ${name} 反引号的方式插入字符串变量
  • s[0]这种赋值方式不起作用，不会改变字符串内容
  • s.toUpperCase()和toLowerCase()返回一个值，
  • s.substring(start,end)返回子串，slice也可以用（Array里面的slice）
  • s.indexOf()

补充：JS中一个字符串中只有数字，如果该字符串前面加了个加号，这个数值就变成了number类型

数组

  • arr.length 获取长度，也可以通过arr.length=值 来改变数组的长度，缺的补undefined，超出的删除
  • 通过索引修改值，索引可以越界，缺的补undefined。但是尽量不要这样做
  • arr.indexOf()
  • arr.slice(start,end) 数组切片，返回一个新数组。如果不加参数返回整个数组，可以用来深拷贝。如果只给一个参数，它将作为起始索引
  • arr.push()和arr.pop() 数组末尾添加、删除。pop()会返回值
  • arr.unshift()和arr.shift() 用来在数组头插入和删除。shift()会返回值
  • arr.sort() 直接对当前的数组进行排序
  • arr.reverse() 颠倒当前数组的元素位置
  • arr.join(‘-‘) 参数必须为字符或字符串，返回一个所有元素以参数相连接的字符串
  • arr.concact( arr2 ) 连接arr和arr2，并返回一个新数组
  • arr.splice(start,num,[item1,item2,…]) 在当前数组上发生改变，从start开始，删除num个元素，在start位置开始插入item。同时将删除掉的元素组成一个数组作为返回值。

对象

  • JavaScript用一个{...}表示一个对象，键值对以xxx: xxx形式申明，用,隔开（最后一个不用加）

  • 如果属性名不是一个有效的变量名，需要用''括起来

  • 访问属性的方式，有效变量名，可以通过person.name或者person[‘name’]来访问，另外的只能通过person[‘middle-school’]访问。访问不存在的属性，不会报错，而是返回undefined

  • 动态类型，可以动态添加、删除属性 。删除：delete person.name，添加：person.age=20 直接赋值即可

  • 判断属性是否存在，两种方式，in判断 和 hasOwnProperty()方法

条件判断

JavaScript使用if () { ... } else { ... }来进行条件判断

循环

方式一：JavaScript使用 for(条件语句){ ... }来实现循环，与C++相同

方式二：for ... in循环，可以把一个对象（注意：不是数组）的所有属性依次循环出来，for..in对Array循环得到的是String

方式三： while() { ... }

方式四： do { ... } while()

如果JS陷入“死循环”，浏览器将无法正常显示，或无法正常执行当前页面逻辑，有的浏览器会直接挂掉

Map和Set

JS中默认的对象{...}格式可以视为C++的Map或Python的dictionary，但是默认的{...}的属性值只能为字符串。为了突破这个限制，ES6新增了Map对象

  • Map定义方式

    • ```javascript

  • Map的函数

Set也是ES6新增的对象，相当于C++的Set

  • Set定义方式

    • ```javascript
      var s = new Set([1,2,’3’]);

  • Set的函数

iterable类型（遍历）

Array可以采用下标循环，但是Map和Set无法使用下标遍历，为了统一，ES6标准引入了iterable类型，Array、Map、Set都属于iterable类型。具有iterable类型的集合可以通过for..of遍历

**还有一种方便的遍历方法——forEach()**，它是iterable内置的方法

实际上，也可以只写一个参数，因为JS的函数调用不要求参数一致

二、函数

定义

  • 有名函数

  • 匿名函数

参数

JS允许传入任意参数，可以多、可以少

  • 传入参数为空

    这种情况，参数x将收到undefined，返回值为NaN。为了避免收到undefined空参，可以在函数内部进行检查

  • 传入参数有多

    arguments，函数内部可用，指向传入的所有参数，类似Array但不是Array。即使函数不定义任何形参，也可以拿到传入的参数值

    rest参数，ES6标准新引入的

    rest前面用...标识，以数组的形式保存，b后面的参数，如果没有，那么rest为空数组。可以用来接受数量不固定的参数

变量作用域

  • 变量在函数内部用var声明，作用域为整个函数体

  • 由于函数可以嵌套，所以内部函数可以使用外部函数的变量。如果同名变量，内部变量覆盖外部变量

  • 一种特殊情况

    因为var声明的变量范围为整个函数体，避免这种情况可以使用let来声明变量 （ES6新增加，let是定义的变量是块级作用域）

  • “变量提升”，JS有一个特性，它会先扫描整个函数体的语句，把变量的声明提升到函数顶部（但是定义并没有提升）

  • 全局作用域，JS有一个默认的全局对象window，所有全局变量绑定为它的一个属性。

  • 命名空间，不同的JavaScript文件如果使用了相同的全局变量，或者定义了相同名字的顶层函数，都会造成命名冲突。为了解决这个问题，可以把所有全局变量和顶层函数绑定到一个全局变量中

一种是省略了{...}和return。一种是不能省略{...}和return的

如果函数参数有多个，需要用()括起来

如果要返回一个对象，也要特别注意


this，箭头函数内部的this是词法作用域，由上下文确定。this总是指向词法作用域，也就是外层调用者obj

由于this已经按照词法作用域进行绑定了，所以call和apply也没有办法修改this指向
