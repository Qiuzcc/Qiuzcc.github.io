笔记——bootstrap框架

bootstrap框架一些常用元素的笔记

一、布局

容器

列布局

相当于flex布局

属性 css bootstrap
主轴对齐方式 justify-content: space-between
justify-content-center
交叉轴对齐方式 align-items: flex-start align-items-start
子元素交叉轴对齐 align-self: start align-self-start

间隙

属性 css bootstrap
内边距 padding=** p-5（水平+垂直） px-5（水平内边距） py-5（垂直内边距）
外边距 margin = ** g-5（水平+垂直） gx-5（水平外边距） gy-5（垂直外边距）

建议在父级容器中添加 overflow-hidden 的类属性

边界border

通过添加 border类名 为容器添加边界

二、文本

文本对齐

.text-center 居中
.text-start 居左
.text-end 居右
.text-sm-center 响应式的对齐方式

自动换行

.text-wrap 自动换行
.text-nowrap 不换行
.text-break 文本断行（防止长单词溢出）

大小写

.text-lowercase 全部转为小写
.text-uppercase 全部转为大写

字体大小

.fs-1 .fs-2 …

字体粗细和斜体

font-weight通用类别的缩写为.fw-*

font-style通用类别的缩写为.fst-*

fw-bold 加粗
fw-bolder 再加粗
fw-light 浅色
fw-lighter 再浅色
fst-italic 斜体

行高

使用.lh-*通用类别来改变行高

lh-base 1.5倍行高
lh-lg 2倍行高

旁注小文本

内联文本元素

引用

响应式图片

通过 Bootstrap 所提供的.img-fluid 类让图片支持响应式布局。其原理是将max-width: 100%; 和 height: auto; 赋予图片，以便随父元素一起缩放。
