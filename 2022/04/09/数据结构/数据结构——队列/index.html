数据结构——队列

语法：template<class T, class Container = deque<T> > class queue

特性：FIFO，先进先出，队尾插入、队头删除

函数

empty 如果队列为空，则该函数返回true，否则返回false
size 返回队列中元素的个数
push 在末尾插入一个新元素，返回值为void
pop 删除第一个元素，返回值为void
front 返回第一个元素
back 返回最后一个元素
